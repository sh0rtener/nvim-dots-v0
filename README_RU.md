# sh0rtener-nvim-dots-v0

## Форматирование и автокод

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `Ctrl + Shift + I`       | Форматирование кода через LSP (асинхронно)     |
| Автосохранение C#        | Перед сохранением запускается `csharpier` для форматирования |

---

## Диагностика и ошибки (LSP)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `<leader> e`             | Показать диагностику/ошибку во всплывающем окне |
| `[ d`                    | Перейти к предыдущей диагностической ошибке     |
| `] d`                    | Перейти к следующей диагностической ошибке      |

---

## NvimTree (файловый проводник)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `<leader> p`             | Открыть / закрыть NvimTree                      |
| `h`                      | Закрыть папку / подняться на уровень выше       |
| `l`                      | Открыть файл                                   |
| `v`                      | Открыть файл в вертикальном сплите             |
| `a`                      | Создать файл                                   |
| `d`                      | Удалить файл / папку                           |
| `r`                      | Переименовать файл / папку                      |
| `c`                      | Копировать файл / папку                          |
| `x`                      | Вырезать файл / папку                            |
| `p`                      | Вставить скопированный / вырезанный файл       |

---

## Буферы и переключение (BufferLine)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `<leader> n`             | Переключиться на предыдущий буфер              |
| `<leader> m`             | Переключиться на следующий буфер               |
| `<leader> 1..9`          | Перейти к конкретному буферу (1–9)            |
| `<leader> q`             | Закрыть текущий буфер                           |

---

## Поиск (Telescope)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `<leader> ff`            | Поиск файлов в проекте                          |
| `<leader> fb`            | Поиск по открытым буферам                       |
| `<leader> fg`            | Поиск текста в проекте (live grep)             |

---

## Отладка (DAP)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `F5`                     | Запуск / продолжить отладку                     |
| `F10`                    | Шаг с обходом (step over)                       |
| `F11`                    | Шаг внутрь (step into)                          |
| `F12`                    | Шаг наружу (step out)                           |
| `<leader> db`            | Поставить / убрать breakpoint                   |
| `<leader> dc`            | Очистить все breakpoints                        |
| `<leader> dr`            | Открыть DAP REPL                                |

---

## LSP: Code Actions и переходы по коду

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `<leader> cl`            | Запустить CodeLens (LSP)                        |
| `<leader> sr`            | Показать все ссылки на символ под курсором      |
| `gd`                     | Перейти к определению символа                  |
| `gD`                     | Перейти к декларации символа                   |
| `gt`                     | Перейти к типу символа                         |
| `gi`                     | Перейти к реализации интерфейсов / методов     |
| `gr`                     | Показать все ссылки на символ                  |
| `K`                      | Показать hover с информацией о символе         |
| `<leader> ca`            | Открыть CodeActions                             |

---

## Навигация между окнами (splits)

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `Ctrl + h`               | Перейти в левое окно                            |
| `Ctrl + j`               | Перейти в нижнее окно                           |
| `Ctrl + k`               | Перейти в верхнее окно                          |
| `Ctrl + l`               | Перейти в правое окно                           |

---

## Insert Mode Aliases

| Клавиша                  | Действие                                        |
| ------------------------ | ----------------------------------------------- |
| `Ctrl + k`               | Выйти из insert mode (Esc)                      |

---

## Комментарии (Comment.nvim)

| Действие                  | Клавиша                                        |
| ------------------------- | ---------------------------------------------- |
| Закомментировать строку     | `gcc`                                          |
| Закомментировать блок       | `gbc`                                          |
| Комментарий выше           | `gcO`                                          |
| Комментарий ниже           | `gco`                                          |
| Комментарий в конце строки | `gcA`                                          |
| Operator-pending комментарий | `gc{motion}` / `gb{motion}`                  |
